# Data Dictionary — llms-authority-ledger

| Field | Meaning |
|---|---|
| `citation_index` | 1-based row number preserving original file order |
| `section_id` | Two-digit section identifier from the ledger (01–24) |
| `section_title` | Section heading/title from the ledger |
| `section_focus` | The section's 'Focus' note (if present) |
| `entity_name` | Name of the citing entity (publication/org/site) |
| `category_type` | Category/type label from the ledger |
| `context_title` | The context/title of the citation/mention |
| `proof_url` | URL pointing to the proof source |
| `source_line` | 1-based line number in llms-authority-ledger_ORIGINAL.txt where the row appears |
